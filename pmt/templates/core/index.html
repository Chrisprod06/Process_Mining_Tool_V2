Welcome to the Process Mining Tool

This web application serves as an initial web interface for the PM4PY Library for process mining!

description Event Logs Handling

Import/Export and modify event logs. Convert as well from CSV to XES and vise versa!

model_training Process Models Handling

Import/Export process models, as well as discover new ones from event logs imported in the system!

checklist Conformance Checking

Perform token based replay and view alignments for process models and event logs!

analytics Performance Dashboard

View various statistics such as total cases, rework and more about an event log imported in the system!

groups Social Network Analysis

View how the staff of your organization work with each other and how much work everyone is doing!

calculate Simulation

Perform simulation such as play out and monte carlo to see the behaviour of your process models!
